# 変更履歴

## [1.0.0] - 2024-10-30

### 追加

- 初回リリース
- Azure Automation を使用した祝日判定付き VM 自動起動/停止機能
- 内閣府 Web サイトからの祝日データ自動取得機能
- 年末年始休暇の自動追加機能
- 完全なハンズオンドキュメント

### 機能

- **add_holidays.ps1**: 祝日データの年次自動更新
- **not_holidays.ps1**: 祝日判定ロジック
- **start_vm.ps1**: VM 自動起動処理
- **stop_vm.ps1**: VM 自動停止処理
- **holiday_automation**: 条件分岐によるスケジュール制御
